# Strikeout Count Generator

A touch-friendly React web app for creating and downloading baseball strikeout-count graphics with forward and backward Ks.

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

Push this folder to GitHub, then import the repository in Vercel. Use the default Vite settings:

- Framework Preset: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
